# MeshMimic Rebuttal Page

This repository hosts the anonymous rebuttal page for the MeshMimic submission.

MeshMimic is a geometry-aware humanoid motion learning framework that couples 3D scene reconstruction with motion learning from monocular videos. It reconstructs terrains and human trajectories, then transfers contact-consistent interactions to a humanoid agent for robust, long-horizon locomotion skills.

The rebuttal page provides supplementary materials, additional experiments, and visual results in response to reviewer comments.
